# 🤖 AI Bot – RAG LLM Chatbot

A simple AI chatbot built using **Retrieval-Augmented Generation (RAG)**. It uses **Ollama** to run a local LLM and provides answers based on retrieved documents.

## 🚀 Tech Stack

- **Frontend:** React, Vite
- **Backend:** FastAPI, Python
- **LLM:** Ollama
- **Deployment:** Render, Streamlit
- **Editor:** Visual Studio Code (VS Code)

## ✨ Features

- AI-powered chatbot
- RAG-based document retrieval
- Fast and responsive interface
- Local LLM with Ollama
- REST API using FastAPI

## 🛠️ Run Locally

```bash
# Clone the repository
git clone https://github.com/your-username/your-repo.git

# Install backend dependencies
pip install -r requirements.txt

# Start FastAPI
uvicorn main:app --reload

# Start frontend
cd frontend
npm install
npm run dev
```

## 📌 About

This project was built to explore how **RAG** and **LLMs** can work together to provide more accurate, context-aware responses. It combines a **React + Vite** frontend with a **FastAPI** backend and uses **Ollama** to run the language model locally.
